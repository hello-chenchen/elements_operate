# elements_operate
Chrome extension for operate DOM elements

#### Version: 0.0.2

#### Function:
Copy element context(rich text) to you clipboard.

Win/Linux:
Use ctrl + shift + 2 to start
Use ctrl + shift + 1 to stop

Mac:
Use cmd + shift + 2 to start
Use cmd + shift + 1 to stop
